Professional Experience

Software Engineering Intern (May 2016 - July 2016).

Bigclozet, Bangalore, India.

About the Organisation

Bigclozet is a women fashion aggregator. Items from different E-Commerce websites are displayed on a single platform.

My work in 9 weeks.
  • Developed multi-threaded crawlers in Python to fetch items from different E-Commerce Websites. Crawlers integrated with kafka framework Producer-Consumer Clients. Integrated Grafana to get better metrics and statistics. Crawlers further improved to update the prices of the items already present in the database. Integrated slack with the crawlers to get notifications regarding crawlers. Testing of crawlers took a lot of time. Catalogue size almost doubled. Crawlers fetched around 1 Million new items.
  • The second task was to write a web service which fetches the live price for an item being viewed by the user. Prices on big E-Commerce websites are changed frequently. Developed a webservice using Node.js, Express and using a java server. Item URL and Seller name were passed to backend which were used to crawl the item page and fetch the live price.
  • The third task was to write RESTful APIs in Node.js and MongoDB to support various functionalities being added to the website and android application.
  • Developed Webview in the android application when the user wants to buy a particular item. Added multiple functionalities in webview to make the user interaction better.
  • Worked on the notification service of the android application. Notifications sent to users notifying them regarding change of price or in-stock information.
  • Major contribution in renovation of the android application for the organization.
Technical Skills learnt/used:
Languages: JavaScript, Java, Python.
Databases: MongoDB
Frameworks/Tools: Apache Kafka, Grafana, Android Studio, Webstorm, PyCharm, IntelliJ IDEA, MongoChef.
