Professional Experience

Software Engineering Intern (May 2016 - July 2016).

Bigclozet, Bangalore, India.

About the Organisation

Bigclozet is a women fashion aggregator. Items from different E-Commerce websites are displayed at one single place. Addition to that many filters, other functionalities are available providing users more options to search.

My work in 9 weeks.
  • Developed crawlers to fetch items from different E-Commerce Websites. The crawlers were written in Python. Crawlers were made multi-threaded to increase the throughput. Backed up with kafka framework to decrease the load on the database server(MongoDB) as many operations used to perform read/write operations on MongoDB. Integrated Grafana to get better statistics of how the crawlers are running(how many new items crawled, for how many prices decreased). Integrated slack with the crawlers so that notifications come up when crawlers failed. Testing of crawlers took a lot of time. Catalogue size almost doubled. Crawlers fetched around 1 Million new items.
  • The second task was to write a web service which fetches the live price for an item being viewed by the user. Prices on big E-Commerce websites are changed frequently. Wrote the service using Node.js, Express and using a java server. Item URL and Seller name were passed to backend which used to crawl the item page and fetch the live price.
  • The third task was to write RESTful APIs in Node.js and MongoDB to support new functionalities being added to the website and android application.
  • Developed Webview in the android application when the user wants to buy a particular item. Added multiple functionalities in webview to make the user interaction better.
  • Worked on the notification service of the android application.
  • Major contribution in renovation of the whole android application of the organization.
