Bingo Web App

Bingo Game implemented as a Web App using Node.js, ExpressJs, HTML and AJAX. Allows matches with Bot and users. Multiple users can login/Register and play at the same time.

Features:

  • Allows User to play with the Bot. Bot implemented at the backend. Bot follows a greedy approach for selection of the next number.
  • User Bingo matrix saved in the backend for computations and checking for BINGO.
  • User Ids cached when user wants to battle with another user. When a match is found, user ids and removed from cache and one user is randomly assigned a chance to play.
  • Client side scripts written to check for the user chance and last move made by the opponent. Loaded implemented till the opponent makes the move.
  • User statistics saved when Game ends. User Data stored and modified in Mongo Database in the backend.

Tools Used:

  • ExpressJs used to server routes and server.
  • Node.js used for backend scripting and Bot gameplay.
  • Client side scripts written in JavaScript, AJAX and JQuery.
  • Bootstrap used for basic UI.
  • Webstorm used as text editor.
  • Git as version control system.

Movie Subtitles Downloader

« »

Developed an Application using Python and subDB API to download synchronised subtitles for a movie.

Features:

  • Terminal based application.
  • Gets the subtitles of the movies passed as arguments to the program. Movie to be present in the same directory as that of the project.
  • Gets first and last 64KB of the video file. Encrypts it and uses it to match the data in the database.
  • Application implemented using SubDB API.
  • After successful download of the subtitle, runs the video file in vlc with the downloaded subtitles.

Tools Used:

  • Git as version control system.
  • SubDB API module for fetching the subtitles.
  • Hashlib module to encrypt the video data.
  • Sublime text as text editor.
