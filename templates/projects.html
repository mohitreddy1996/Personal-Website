Face Detection Bot

« »

A Raspberry Pi 2 bot which detects a face and follows it using Computer vision and Image processing Algorithms.

Features:

  • Image Processing and computer vision algorithms written in Python.
  • Raspberry Pi 2 and Camera module of RP2 used for capturing the faces.
  • GPIO Module of Python for Raspberry Pi 2 used to control the motors.
  • Simple Algorithm to detect the movement of the face and required motor to run.

Tools Used:

  • Putty to connect to Raspberry Pi.
  • VNC Server to use Laptop as the screen.
  • Git for version control system.
